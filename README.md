# pine.js

Built to overcome the language limitations of TradingView's Pine scripting language. Written in TypeScript and Node.Js, pine.js an extensible trading platform supports custom expressions, live time series valuation, generic trading interface. The goal of this platform is to be able to execute machin learning models for trading signals and automate trade executions.
